Simple implementation of kafka stream processing 'koober' example.

### Technology Stack:
* Micronaut
* Java 11
* Angular 9
* Kafka
* Docker

### How to run demo:
1. docker-compose up -d
2. Go to ```localhost:4200/driver```
3. Go to ```localhost:4200/rider``` and click on the map
